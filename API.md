# API

## 食堂

* 总体情况：https://canteen.sjtu.edu.cn/CARD/Ajax/Place
* 各餐厅具体情况：https://canteen.sjtu.edu.cn/CARD/Ajax/PlaceDetails/$(x)00
  * 其中x(一到七餐)=1~7，x(哈乐)=8，x(玉兰苑)=9。
